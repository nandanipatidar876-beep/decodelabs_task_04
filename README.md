# decodelabs_task_04

## Description
The General Knowledge Quiz is a simple Python console application that tests a user's basic knowledge through three general knowledge questions. The program takes user input, checks answers using if-else statements, and awards one point for each correct answer. At the end of the quiz, the total score is displayed along with a performance message.

## Features
- Asks 3 general knowledge questions
- Accepts and evaluates user responses
- Uses if-else logic for answer checking
- Maintains a score counter
- Displays the final score and feedback

## Technologies Used
- Python 3
- Variables
- If-Else Statements
- User Input Functions

## Learning Outcomes
This project helps beginners understand control flow, user interaction, conditional statements, and score management in Python. It is a great introduction to building interactive console-based applications.
